## G2S PokeCrystal->PokeEmerald Tileset Package
- Ported by Xyifer largely by hand, organized and cleaned up with Porytiles, helped greatly by Paint.NET and Polished Map and GraphicsGale.

This is my set of Pokemon Crystal (USA) tilesets formatted for Pokemon Emerald, including ancillary files such as optional palettes and a Porymap prefab file for that awfully annoying Ice Path.

A bit of work is required as tile attributes are not yet completely set, but graphically this is ready to use.
