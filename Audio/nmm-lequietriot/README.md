How this folder is structured

---

Currently, there is only 1 folder, which has the unique "FR/LG Style" MIDI files, meant for a base pokefirered project. Included within, is also a voice\_groups\_addon-snippet.txt file, which is meant to be added at the end of an existing voice\_groups.inc from pokefirered. Again, this is provided as-is, and may require tweaking to match whatever naming you have. If you choose to use the voicegroup addons, make sure to fully add 191, 192, and 193. The main voicegroup for the midis will always be 192. There is also another readme file in the folder if you want to see specific notes on each remix.
