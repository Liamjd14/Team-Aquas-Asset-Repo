Installation is like any other song, nothing special.
You MUST use the `voicegroup192.inc` and the `drums` from my `Jazzy Song` folder in the sibling directory in Github.
AND ALSO modify your `direct_sounddata.inc` with the `new_direct_sounddata.inc` as well.
This song uses the same drums as Jazzy Song.

1. Copy the `drums` folder from `Specker/Jazzy Song` in the sibling folder to your DirectSound directory
    - You must also add to your `direct_sound_samples.inc`.
    - *If* you're hard-headed enough, 
2. Include the new `sadpop.inc` voicegroup in your `voice_groups.inc` file.
3. Add sadpop.midi to your `midi.cfg` at a nice volume/reverb.
4. Add a reference to the song in `songs.h` and `debug.h`.

Feel free to reach out to `specker1` if you have any trouble.
